# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Workspace

This is a general Claude Code workspace directory, not tied to a specific project. Individual projects should be opened from their own directories.

## Configured Permissions

The following commands are pre-approved in `.claude/settings.local.json`:
- `gh auth *` (Bash)
- `gh auth status` (PowerShell)
